<h2 align="center">Installation</h2>

1. Fork and clone this repository using  

```
  git clone <repo link>
  cd <repo>
```  

2. Configure and run the backend (you need to have NPM installed on your machine)

```
  cd <API Folder>
  npm install
  npm start
```
